# Инструкция по начальной настройке

## Дополнительная настройка новых разделов "Обслуживание клиентов (ВКМ)" и "Расчет управленческой зарплаты (ВКМ)"


1. Для раздела "Обслуживание клиентов (ВКМ)" необходимо:

1.1. В группе "Справочники" в справочнике "Сотрудники" создать карточки сотрудников
1.2. В разделе "Администрирование" создать пользователей информационной базы, в соответствии с ранее созданными сотрудниками:
     - сотрудникам категории "Специалист" выбрать права доступа "Специалист"
     - сотрудникам категории "Менеджер" выбрать права доступа "Менеджер" 
     - сотрудникам категории "Бухгалтер IT-Фирмы" выбрать права доступа "БухгалтерИТФирмы"
     - сотрудникам категории "Кадровик-расчетчик" выбрать права доступа "Кадровик расчетчик"
1.3. Заполнить константы "Номенклатура абонентская плата" и "Номенклатура работы специалиста":
     - в группе "Справочники" в справочнике "Номенклатура" создать карточки номенклатуры, которые будут отражаться в константах для дальнейшего оформления документа "Реализация товаров и услуг"
     - в группе "Сервис" заполнить константы "Номенклатура абонентская плата" и "Номенклатура работы специалиста", в соответствии с ранее созданными номенклатурами
1.4. Заполнить константы "Токен управления телеграм-ботом" и "Идентификатор группы для оповещения":
     - создать в телеграме бота, сохранив токен в константе "Токен управления телеграм-ботом"
     - создать в телеграме группу, добавьте туда созданного бота и всех сотрудников, сохранив идентификатор группы в константе "Идентификатор группы для оповещения"


2. Для раздела "Расчет управленческой зарплаты (ВКМ)" необходимо:

2.1. В группе "Справочники" в справочнике "Графики работы" создать карточки графиков работы
2.2. В группе "Регистры" необходимо заполнить регистр "Режим работы", для этого необходимо воспользоваться обработкой "Заполнение режима работы", которая расположена в группе "Сервис" 
2.3. В группе "Регистры" необходимо заполнить регистр "Условия оплаты сотрудников"
